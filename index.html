The idea.

Most meditation apps sell a single technique — focused breathing — repackaged a hundred ways. The Vijñāna Bhairava Tantra, written around 800 CE, is something stranger: a catalogue of 112 distinct techniques, each a different doorway into the same stillness.

Patañjali's Yoga Sūtras add the missing layer — which inner gesture meets which state (1.33's four keys; 2.33's pratipakṣa-bhāvana, cultivating the opposite).

Spanda treats that ancient corpus as what it functionally is: a lookup table from human state → precise method. You say where you are. It opens the door built for that exact place.
